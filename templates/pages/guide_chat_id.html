Telegram Chat ID Guide

Obtain your unique Chat ID to receive instant notifications from your portfolio bot.

1 Start Your Bot

Open a chat with the bot you just created and click "Start" or send /start.

2 Find ID Bot

Search for @userinfobot or @GetMyIDBot in Telegram.

3 Get Your ID

Start a chat with the ID bot. It will immediately reply with your User ID (a series of numbers like 123456789).

4 Input in Dashboard

Copy this number and paste it into the Telegram Chat ID field in your dashboard settings.
